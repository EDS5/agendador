Objetivo:
Criar um software que ajude a gerenciar e agendar os espaços físicos do CCS que são reservados pediódicamente via e-mail ou então formulários web do CCS.


Software web

Tecnologias: PHP, MySQL, HTML, CCS, Zend Framework.


Update: Python + Django + MySQL


Levantamento de requisitos:

- Página de abertura;
- Listar espaços físicos;
- Formulário de reserva para cada espaço físico;
- Formulário para cancelar reserva de espaço físico;
- Controle de choque de horários e prioridades;
- Área de administração;
- E-mail de confirmação de reserva;
- Definir quais tipos de eventos acontecem em quais locais;
- ...
